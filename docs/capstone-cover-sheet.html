Project Summary

Kistie Store is a full-stack e-commerce application for a women's fashion boutique based in Kampala, Uganda. The platform lets customers browse premium imported fashion (Turkey, UK, USA), filter by category and EU size, add items to a session-based cart, view totals in four currencies (USD, EUR, KES, UGX), and complete checkout via staff-verified payment (MTN Mobile Money, Airtel Money, WorldRemit). Store staff manage inventory, orders, and customer inquiries through a custom-themed Django admin panel and a staff dashboard. The project is deployed on Render with PostgreSQL and serves as the capstone demonstration of full-stack Django development.
