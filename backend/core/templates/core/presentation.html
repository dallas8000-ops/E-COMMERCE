Home Page

URL: `/`

Purpose: Welcome visitors and provide an overview of the store and its offerings.

User Actions Available:

  • Get an overview of the store and its product categories
  • Use the top navigation bar to jump to any section of the site
  • Click Inventory in the navbar to start browsing products
  • Click About to read the brand story and mission
  • View the hero image showcasing the brand aesthetic

Key Features: Responsive hero section, prominent call-to-action buttons, mobile-friendly navigation menu.

About Page

URL: `/about/`

Purpose: Communicate the company's mission, values, vision, and brand story.

User Actions Available:

  • Read about the company mission, values, and product focus
  • Learn the brand ethos and vision statement
  • View brand imagery and store aesthetic
  • Access contact information (phone, WhatsApp, location)
  • Use the navbar to navigate to Inventory to start shopping
  • Use the navbar to navigate to Cart to continue an order

Inventory Page

URL: `/inventory/`

Purpose: Display all available products for browsing and selection.

User Actions Available:

  • Browse the full product catalog loaded from the backend
  • View product images, names, and prices on each card
  • See stock status for each product
  • Click Add to Cart on any product card to add that item to your current order
  • Navigate back using the navbar links
  • Navigate to Cart in the navbar when ready to review and pay

Key Features: Product grid layout (3 columns on desktop, 1 on mobile), product card details, instant add-to-cart functionality, dynamic product loading from database.

Cart Page

URL: `/cart/`

Purpose: Allow customers to review their order, select currency and payment method, and proceed to checkout.

User Actions Available:

  • Review all items currently in your order (name, unit price, quantity, line total)
  • Choose your preferred currency from the dropdown: USD, EUR, KES, or UGX
  • View totals automatically updated using live exchange rates
  • Select a payment method: MTN Mobile Money, Airtel Money, or Pesapal
  • Review rate source (Live API or Fallback) and last updated timestamp
  • Click Pay Now to submit the order and receive a payment confirmation or reference number
  • Update currency and payment method to recalculate totals

Key Features: Multi-currency support with live exchange rates, payment method selection, order summary, responsive form controls, real-time total calculation.

Django Admin

URL: `http://127.0.0.1:8000/admin/`

Purpose: Secure backend for store administrators to manage inventory, orders, and customer data.

Admin Actions Available:

  • Log in with superuser credentials for secure access
  • Products — Add, edit, or delete products (name, price, category, stock quantity, images)
  • Categories — Create and manage product categories
  • Product Images — Upload and manage product images inline with product records
  • Carts / Orders — View all customer carts and placed orders; update order status (e.g., Pending → Payment confirmed)
  • Contact Inquiries — Read messages submitted via the contact form
  • Users / Groups — Manage shopper accounts and staff permissions
  • Export data and generate reports as needed

Key Features: Django's built-in admin interface, role-based access control, inline editing, search and filtering, bulk actions.

Responsive Features Implemented

✓ Core Responsive Features

  • Bootstrap 5 Grid System — 12-column responsive grid adapts across all breakpoints
  • Mobile-First Design — Navigation collapses to hamburger menu on mobile devices
  • Flexible Typography — Font sizes and line heights scale appropriately for readability
  • Responsive Images — Product images scale with container using CSS flexbox and object-fit
  • Touch-Friendly Controls — Buttons and form inputs sized for easy interaction on mobile
  • Adaptive Layout — Product grids: 3 columns desktop → 1 column mobile
  • Fluid Spacing — Padding and margins adjust based on viewport width
  • Viewport Meta Tag — Ensures proper scaling on mobile browsers
  • CSS Media Queries — Custom breakpoints for optimal layout at each device size

Layout Adjustments by Screen Size

📱 Mobile (375px)

  • Single-column layout for all pages
  • Hamburger menu for navigation (collapsed navbar)
  • Full-width product cards with vertical stacking
  • Stacked form controls and input fields
  • Large touch targets for buttons and links
  • Simplified data tables with horizontal scrolling
  • Images scaled to container width

📱 Tablet (768px)

  • Two-column layout where appropriate
  • Full navigation menu visible
  • Product grid with 2 columns
  • Balanced whitespace with adequate padding
  • Forms display side-by-side in rows
  • Larger readable text without excessive sizing

🖥️ Desktop (1440px)

  • Side-by-side form controls for efficient data entry
